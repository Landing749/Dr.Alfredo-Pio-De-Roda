Edge or Chrome may flag the download with a message like "This file isn't commonly downloaded" or show it greyed out at the bottom of the window.

Setup walkthrough

Installing ATSYS

The same installer handles a first-time install and a machine that already has ATSYS on it — it detects which situation you're in automatically.

First time on this computer

  1. Download the installer

    Save the installer.exe file anywhere on your computer, e.g. your Downloads folder.

  2. Run it

    Double-click the file. No administrator rights are required. If Windows SmartScreen appears, click More info → Run anyway — see above for details.

  3. Accept the license agreement

    Read the EULA and select I accept the agreement to continue.

  4. Choose a desktop icon (optional)

    Tick the box if you'd like a shortcut on your desktop, then click through to install.

  5. Finish and open the Welcome guide

    When setup completes, it can immediately open the post_install.html guide and launch ATSYS for you.

  6. Enter your license key

    On first launch, ATSYS asks for a license key — enter it exactly as your school was given it. It's validated once and remembered after that.

ATSYS is already installed

Run the same installer file again (with SmartScreen → More info → Run anyway, same as above). It recognizes the existing install and opens a small panel titled "ATSYS is already installed" with three buttons instead of the full setup wizard — click the top one, "Update", unless you have a specific reason to repair or remove:

Recommended

Update

Replaces the program files with the new version only. Skips the EULA, folder, and options pages entirely. Your settings, data files and license are untouched.

If something's broken

Repair / Reinstall

Reinstalls all program files and restores shortcuts. Use this if a shortcut is missing or the app won't open properly.

Remove

Uninstall

Removes ATSYS completely after a confirmation prompt. Your SF2 files and saved data are kept — only the program itself is removed.

Close ATSYS before running the installer. If it's still open, setup will ask to close it automatically so it can safely replace files that Windows locks while the app is running.
Common questions

FAQ & troubleshooting

The most common issues teachers run into, straight from the ATSYS support documentation.

Do I need to uninstall the old version first?+
No. Just run the new installer over the old install — it detects ATSYS is already present and offers Update, Repair, or Uninstall instead of the full setup wizard. Update is the option most people want.
Where do I get a license key?+
Your school is issued a license key by ATH Studios. Enter it exactly as provided on first launch — it's case-sensitive. If you see "License already in use on another device," the key is locked to a different computer; contact ATH Studios to move it.
ATSYS says "No SF2 file found in Active folder"+
ATSYS looks for one .xlsx file in ~/SF2_Files/Active/. Copy your SF2 Excel file into that folder, or use Import File from the Dashboard. Only keep one file in that folder at a time.
"Camera not found" or the QR code never scans+
First check Windows camera privacy: Settings → Privacy → Camera, and make sure both "Allow apps to access your camera" and the desktop-apps toggle are on — this is the most common cause on school-managed machines. Also close Teams, Zoom, or any other app that might be holding the camera. If the camera opens but nothing scans, check lighting and hold the QR card 15–40 cm from the lens.
Antivirus is blocking or quarantining files+
Some antivirus tools flag Python's native camera/QR components as suspicious. Add the ATSYS install folder and ~/SF2_Files/ to your antivirus exclusions (on Windows Defender: Virus & Threat Protection → Manage Settings → Exclusions).
The EULA screen keeps reappearing every time I open the app+
The acceptance marker file couldn't be saved, usually because ~/.attendance_system/ sits on a network drive or a OneDrive-synced folder that isn't always available. Make sure that folder is on your computer's local disk.
Is my attendance data backed up?+
Yes. Every scan is written to a Scan Journal CSV before ATSYS even attempts to save the Excel file, so a scan is never lost even if the SF2 save fails. Timestamped backups are also kept in ~/SF2_Files/Backups/.
Can I install this on a Mac or Chromebook?+
No — ATSYS is Windows-only. Several features depend directly on Windows system APIs and have no equivalent on macOS, Linux, or ChromeOS.
The installer won't run, or Windows blocks it+
Confirm you're on Windows 10 or later, 64-bit — the installer refuses to run on anything older. On IT-managed machines, AppLocker or a Software Restriction Policy may need an exception added by your IT admin for the ATSYS executable path.
