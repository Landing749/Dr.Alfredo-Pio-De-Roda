Before You Install
System Requirements
Windows 10 or 11
64-bit only. Not compatible with Windows 7 or 8.
Webcam / USB Camera
Required for QR code scanning. Any standard camera works.
Internet Connection
Needed for license validation and SF2 portal uploads.
License Key Required
Contact your administrator to receive a valid license key on first run.
Getting Started
How to Install
1
Download the file
Click the Download button above. The file will open in Google Drive — click the download icon (⬇) at the top right of that page.
2
Allow Windows to run it
Windows may show a "Windows protected your PC" warning. Click More info, then click Run anyway. This is normal for new software.
💡 This warning appears because the app is not yet registered with Microsoft. The file is safe — it was built and distributed directly by the school.
3
Accept the license agreement
On first launch, you will be shown the End User License Agreement (EULA). Read and accept to continue.
4
Enter your license key
Enter the license key provided by your administrator. This is a one-time step — the key is saved permanently on your device.
5
Load your class Excel file and start scanning
Click Load Excel File, select your class list, then click Start Camera to begin taking attendance.
